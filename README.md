# Travel Explorer Website

## Project Overview

**Travel Explorer** is a responsive travel website that helps users explore destinations, view itineraries, read reviews, browse photo galleries, and submit feedback.

This project was developed as part of a Web Development assignment using **HTML5, CSS3, and basic JavaScript**.

---

## Site Map

* `home.html` – Home page
* `destinations.html` – Travel destinations
* `itinerary.html` – Travel plans
* `reviews.html` – User reviews
* `gallery.html` – Photo gallery
* `contact.html` – Feedback form
* `map.html` – Maps & videos

---

## Features

### Navigation

* Consistent navigation bar across all pages
* Internal links between pages

### Media

* Multiple images with `alt` text
* Gallery with `figure` & `figcaption`
* Embedded Google Maps & YouTube videos using `iframe`

### Form (Full-featured)

* Text, email, tel, number, date inputs
* Radio buttons & checkboxes
* Select with `optgroup`
* Textarea & datalist
* Hidden input
* Submit & reset buttons
* Validation using `required`, `pattern`, `min/max`
* JavaScript alert on submission

### Design & Styling

* External CSS file
* Flexbox & Grid layout
* Responsive design
* Hover effects & transitions
* Custom fonts using `@font-face`
* Consistent color theme

### Accessibility

* Semantic HTML (header, nav, main, section, article, footer)
* Proper heading structure
* Alt text for all images
* Readable font sizes

### Optimization

* Organized folder structure
* Favicon included
* Meta tags (viewport, description)
